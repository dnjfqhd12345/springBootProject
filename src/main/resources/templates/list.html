전산쟁이의 개발실에 오신 것을 환영합니다!


안녕하세요! <전산쟁이 개발실> 커뮤니티를 만든 전산쟁이입니다!!

프로그래머나 프로그래밍에 관심이 있는 사람들이 모여 편안하게 서로 이야기를 나눌 공간을 만들어보고 싶어서 웹사이트를 제작하게 되었습니다.

이 웹사이트는 자신이 작성한 코드 대해서 어디서 잘못 되었는지 피드백을 받거나 서로 토론하거나 프로그래밍 하다가 지쳤을 때 자유롭게 수다를 떨 수 있는 편안한 공간이라고 생각하시면 됩니다.

오늘도 웹사이트를 방문해주셔서 감사합니다.

웹페이지 기술 스택: Spring Boot, Maria DB, JPA, Thymeleaf, Bootstrap, CKEditor4

 

<전산쟁이의 개발실> 커뮤니티 이용수칙입니다.

1. 각 게시판의 성격에 맞게 글을 게시해주세요.

2. 편안한 공간이 될 수 있도록 서로 배려해주세요.

3. 분쟁이나 논란이 되는 글을 작성 시에 통보없이 삭제/탈퇴 처리가 될 수 있어요.

4. 이미지는 URL 형식으로만 올리실 수 있어요. (DB 용량이 적어서 죄송합니다 ㅜ)
